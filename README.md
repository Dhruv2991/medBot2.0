# 🏥 MedBot 2.0

An AI-powered healthcare assistant that provides:

* 🧠 Medical triage using machine learning
* 🏥 Hospital department recommendations
* 🚑 First aid guidance for emergencies

## 🚀 Features

* ML-based symptom analysis
* Real-time hospital routing
* First aid chatbot
* Clean frontend UI

## 🛠 Tech Stack

* Python (Flask)
* Machine Learning (Scikit-learn)
* HTML, CSS, JavaScript
* REST APIs

## ▶️ How to Run

### Backend

```bash
cd backend
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python -m hospital.hospital_server
```

### Frontend

Open:

```bash
frontend/index.html
```
